# Documentation

Welcome to the documentation site. This site provides guides and reference material to help you get started and make the most of available features.

## Overview

This documentation is organized into the following sections:

- **[Getting Started](docs/getting-started)** – Set up your environment and get up and running quickly.
- **[Usage](docs/usage)** – Learn how to use the core features.
- **[Configuration](docs/configuration)** – Customize and configure the project to suit your needs.
- **[Architecture](docs/architecture/overview)** – System design, container diagrams, and data-flow diagrams.
- **[API Reference](docs/api/overview)** – REST API conventions, authentication, and endpoint catalogue.
- **[Guides](docs/guides/getting-started)** – Installation, configuration, and platform getting-started tutorials.
- **[Development](docs/development/setup)** – Local dev setup, contributing guidelines, and testing practices.
